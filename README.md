# Ameer Muavia Shah - Senior Software Engineer Portfolio

## Overview
This repository contains the complete source code for the personal portfolio of Ameer Muavia Shah, a **Senior Software Engineer** specializing in **ERPNext**, the **Frappe Framework**, and **Python** automation, with over 5 years of enterprise experience.

The portfolio is designed to be fast, responsive, and easy to maintain, leveraging Tailwind CSS for utility-first styling and vanilla JavaScript for dynamic content, particularly for the Projects section.

## üåü Key Features

* **Senior-Level Expertise:** Content is tailored to highlight advanced skills in custom module development, workflow design, and technical mentorship.
* **Dynamic Project Loading:** Project details (including results, links, and roles) are stored in a centralized JavaScript object (`script.js`) and dynamically rendered onto the `projects.html` page and into the interactive modal.
* **Performance & SEO:** Utilizes asynchronous loading for the Tailwind CDN and includes necessary `robots` meta tags.
* **ERP/Frappe Specialization:** Clearly structured content focusing on enterprise solutions, ERP, and automation skills.
* **Modern Design:** Sleek, dark-mode theme built with the Tailwind CSS CDN.
